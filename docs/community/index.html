Join the Butler community

Butler is an open source project that anyone in the community can use, improve, and enjoy. We'd love you to join us! Here's a few ways to find out what's happening and get involved.

Learn and Connect

Using or want to use Butler? Find out more here:

  • SenseOps mailing list: SenseOps newsletter from Ptarmigan Labs
  • Discussion forums on GitHub: Talk about both developing and using Butler Auth.
  • Ptarmigan Labs on Twitter: Follow us here for latest news
  • LinkedIn: Reach out to Ptarmigan Labs via LinkedIn

Develop and Contribute

If you want to get more involved by contributing to Butler, join us here:

  • GitHub: Development takes place here!
  • Bug reports & Feature requests: Let us know what's working, what's not and what's missing.
  • Get in touch with Ptarmigan Labs: Get in touch with Ptarmigan Labs around Butler

You can find out how to contribute to these docs in our Contribution Guidelines.
